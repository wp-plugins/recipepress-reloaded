== Comment to Hungarian translation in English in English == 

(A magyar változatot az angol alatt találja.)

From RecipePress reloaded release 0.X.X the pluging comes with Hungarian translation.

Translator is Tamás Boros.

Please note that after installing the plugin in a Hungarian Wordpress environment, some of the recipe terms still appear in English.

To have everything in Hungarian you need to update some of the metadata managed by Wordpress directly.

Step 1: Install the RecipePress reloaded plugin from the Wordpress Admin menu "Bővítmények" item. 
Step 2: In the Wordpress Admin menu find the "Receptek" item. Open it.
Step 3: Find "Beállítások", within that: "Recept metaadatok" Open it.
Step 4: Find "Egyéni besorolások kezelése", click on the "Egyéni besorolások kezelése" (Manage customer tags) button.
Step 5: You find a list of taxonomy items. Edit all which is not Hungarian.

The Hungarian names to enter:

Címke		Név egyesszámban	Név			Keresőbarát név (slug)	
---------------------------------------------------------------------------------------
post_tag	Címke			Címkék			cimke
rpr_ingredient	Hozzávaló		Hozzávalók		hozzavalo	 
rpr_course	Fogás			Fogások			fogas	
rpr_cuisine	Konyhastílus		Konyhastílusok		konyhastilus	 
rpr_category	Recept kategória	Recept kategória 	rpr_category (this slug can't be Hungarian)
rpr_tag		Recept címke		Recept címkék		rpr_tag (this slug can't be Hungarian)

To apply the changes:

(a) Under Műveletek, click on "Szerkesztés" button
(b) In the window enter the Hungarian terms
(c) When done click on "Mentés" (Save)

Do it for all rows as needed.

Then you can exit from the menu.

Now you should see all RecipePress reloaded public and admin terms in Hungarian. By chance , if not, just refresh the page.


== End of Comment to Hungarian translation in English ==

== A magyar fordításhoz fűzött megjegyzések ==

A RecipePress reloaded 0.X.X kiadástól kezdve a bővítmény magyarul is használható.

Fordította Boros Tamás.

Figyelem: a bővítményt magyar Wordpress környezet alatt telepítve a megnevezések többsége automatikusan magyar nyelvű lesz, azonban lesz néhány kivétel.

Ahhoz, hogy minden magyarul legyen, néhány metaadatot, melyet közvetlenül a Wordpress kezel, önnek kell magyarra frissítenie.

1. lépés: A Wordpress Admin menü "Bővítmények" menüjéből a telepítse a RecipePress reloaded bővítményt
2. lépés: A baloldoldali Wordpress Admin menüben nyissa meg a "Receptek" menüt.
3. lépés: Kattintson a "Beállítások" elemre, ezen belül a "Recept metaadatok"-at nyissa meg.
4. lépés: Az "Egyéni besorolások kezelése" alatt kattintson az "Egyéni besorolások kezelése" (Manage customer tags) gombra.
5. lépés: Látható a besorolási elemek listája. Frissítse azokat, melyek nem magyarul jelennek meg.

A beírandó magyar megnevezések:

Címke		Név egyes számban	Név			Keresőbarát név (slug)	
---------------------------------------------------------------------------------------
post_tag	Címke			Címkék			cimke
rpr_ingredient	Hozzávaló		Hozzávalók		hozzavalo	 
rpr_course	Fogás			Fogások			fogas	
rpr_cuisine	Konyhastílus		Konyhastílusok		konyhastilus	 
rpr_category	Recept kategória	Recept kategória 	rpr_category (ez nem fordítható magyarra)
rpr_tag		Recept címke		Recept címkék		rpr_tag (ez nem fordítható magyarra)

A változások elvégzéséhez:

(a) A "Műveletek" alatt kattintson a "Szerkesztés" gombra
(b) Az ablakban az egyes mezőkbe írja be a magyar megnevezéseket.
(c) Amikor befejezte, kattintson a "Mentés" (Save) gombra.

Hajtsa végre ezt a három lépést a lista valamennyi elemére.

Ezután kiléphet a menüből.

Most már minden, a RecipePress reloaded-hez tartozó megnevezés magyarul jelenik meg. Ha esetleg mégsem, akkor egyszerűen csak frissítse a web-oldalt.
